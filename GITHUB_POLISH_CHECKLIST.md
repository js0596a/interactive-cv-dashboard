## 2) Pin the right 6 repositories

Pin these first:

## 3) Make each pinned repo recruiter-readable

For each pinned repo:

- Add a one-line description in GitHub settings

- Add one screenshot or visual preview

## 4) Keep naming clean

- Use clear repo names (no random suffixes)
- Use title case in README headers
- Avoid empty repos in pinned section
